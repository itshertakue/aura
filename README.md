# Aura Carbon Intelligence 🌌
**Autonomous Intelligence for the Green Economy**

Aura is a premium carbon intelligence platform built for the **Orion Build Challenge 2026**. It transforms complex climate data into actionable financial insights, helping enterprises navigate the transition to a net-zero future with autonomous precision.

![Aura Dashboard](public/screenshot_dashboard.png) *(Placeholder for your demo screenshot)*

## ✨ Core Features

### 🤖 Auto-Reader (AI Document Ingestion)
Eliminate manual data entry. Aura's Auto-Reader uses advanced AI to ingest logistics invoices, utility bills, and supplier reports, automatically extracting carbon intensity and financial metadata.

### 🗺️ Impact Map (Global Visualization)
Visualize your footprint across the entire supply chain. Identify emission hotspots in real-time and understand the environmental impact of every logistics node.

### 🚀 Climate Copilot (Conversational ROI)
Your autonomous strategic advisor. Ask Aura how to reduce your carbon tax liability, optimize shipping routes, or select green vendors. Aura doesn't just show data; it provides intelligence.

### 💸 Tax-Optimizer
Real-time tracking of carbon tax liabilities and potential rebates. Aura identifies optimization paths that turn climate compliance into a competitive advantage.

## 🚀 Getting Started

1. **Clone & Install:**
   ```bash
   git clone [your-repo-url]
   npm install
   ```

2. **Environment Setup:**
   Create a `.env.local` file and add your Google Generative AI Key:
   ```env
   GOOGLE_GENERATIVE_AI_API_KEY=your_key_here
   ```

3. **Run Dev Server:**
   ```bash
   npm run dev
   ```
